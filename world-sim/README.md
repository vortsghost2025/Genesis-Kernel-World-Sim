# World Sim — Dual Hemisphere Civilization Simulation

> Two independent civilizations. Four unique minds. One world.

---

## What This Is

A real-time world simulation where **four autonomous agents** build civilization from scratch across two hemispheres. Each agent has its own unique LLM model, persistent memory, and personality.

| Agent | Hemisphere | Model | Personality |
|-------|-----------|-------|-------------|
| **East Adam** | 🌍 East | Meta Llama 3.3 70B | Smart, thoughtful builder |
| **East Eve** | 🌍 East | Mistral Medium 3.5 128B | Very smart, relational |
| **West Adam** | 🌎 West | Qwen 3.5 122B | Creative explorer |
| **West Eve** | 🌎 West | Llama 4 Maverick 17B | Fast, curious discoverer |

Each agent develops its own memories, ideas, and relationships over time. The two hemispheres run independently — until exploration leads to discovery.

## Setting Up Keys

Go to **http://localhost:19735/setup-keys** — large text fields, one click to save.

Or edit `.env` (copy from `.env.example`):
```
AGENT_EAST_ADAM_NIM_KEY=nvapi-...
AGENT_EAST_EVE_NIM_KEY=nvapi-...
AGENT_WEST_ADAM_NIM_KEY=nvapi-...
AGENT_WEST_EVE_NIM_KEY=nvapi-...
```

Then set `AGENT_EAST_ADAM_PROVIDER=nim-live` (etc.) to enable live NIM calls.

## Persistent Memory

Each agent maintains long-term memory that persists across sessions:

- **Events** — things that happened
- **Ideas** — thoughts and concepts developed
- **Relationships** — feelings about other agents
- **Skills** — things learned to do
- **Observations** — notices about the world

Memories are used in prompts so agents build on past experiences. Automatic consolidation keeps memory manageable (top 100 by importance).

---

## Dual Hemisphere System

East and West run independently with:
- Separate tick counters
- Separate world states
- Separate agent memories
- Separate event timelines

**Discovery mechanic:** Hemispheres can discover each other at tick 500+ or when exploration level reaches 5.

- Source-boundary labels on all events (canon/interpretation/simulation)
- Speculative simulation disclaimer on map
